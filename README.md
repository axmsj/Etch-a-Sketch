# Etch-a-Sketch

![OpenSSF Scorecard](https://api.securityscorecards.dev/projects/github.com/axmsj/Etch-a-Sketch/badge)

A browser-based sketchpad inspired by the classic Etch A Sketch toy, built with vanilla HTML, CSS, and JavaScript. Hover over the grid to draw.

## Demo

Clone the repo and open `index.html` in a browser — no build step required.

## Features

- Interactive grid that responds to mouse hover
- Adjustable grid resolution
- Clear/reset the canvas
- Pure front-end, no dependencies

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## Getting Started

​```bash
git clone https://github.com/axmsj/Etch-a-Sketch.git
cd Etch-a-Sketch
open index.html
​```

Or drag `index.html` into your browser.
